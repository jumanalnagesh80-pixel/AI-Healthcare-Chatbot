## 🔧 TROUBLESHOOTING

### Issue: Python not installed
**Fix:** Download from https://www.python.org/downloads/

### Issue: Missing files after download
**Fix:** Download from `ready-to-use-final` branch specifically

### Issue: Port 5000 in use
**Fix:** Edit `app_modern.py`, change port to 5001

### Issue: Module not found errors
**Fix:** `pip install -r requirements_modern.txt`

### Issue: Database errors
**Fix:** 
```bash
rm healthcare.db
python setup_enhanced.py
python add_health_goals_table.py
```
