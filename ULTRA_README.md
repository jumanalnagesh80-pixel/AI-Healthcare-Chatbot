## 🚀 **Quick Start (Mac M2)**

### **Installation**
```bash
# Clone the ultra-advanced version
git clone -b ultra-advanced-professional https://github.com/jumanalnagesh80-pixel/AI-Healthcare-Chatbot.git
cd AI-Healthcare-Chatbot

# Install dependencies
pip3 install -r requirements_ultra.txt

# Run the application
python3 app_ultra.py
```

### **Access**
- **User Portal**: http://127.0.0.1:5000
- **Admin Portal**: http://127.0.0.1:5000/admin
