═══════════════════════════════════════════════════════════════
    🚀 HOW TO RUN (After Download)
═══════════════════════════════════════════════════════════════

FOR WINDOWS USERS:
──────────────────

1. Install Python from python.org (if you don't have it)

2. Open Command Prompt (cmd):
   - Press Win+R
   - Type "cmd"
   - Press Enter

3. Navigate to project folder:
   cd C:\Users\YourName\Downloads\AI-Healthcare-Chatbot-ready-to-use-final

4. Install dependencies:
   pip install -r requirements_modern.txt

5. Setup database:
   python setup_enhanced.py
   python add_health_goals_table.py

6. Run application:
   python app_modern.py

7. Open browser:
   http://localhost:5000


FOR MAC/LINUX USERS:
────────────────────

1. Open Terminal

2. Navigate to project folder:
   cd ~/Downloads/AI-Healthcare-Chatbot-ready-to-use-final

3. Run automated setup:
   bash SETUP_EASY.sh

   OR Manual setup:
   pip3 install -r requirements_modern.txt
   python3 setup_enhanced.py
   python3 add_health_goals_table.py
   python3 app_modern.py

4. Open browser:
   http://localhost:5000

═══════════════════════════════════════════════════════════════
    🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

PROBLEM: "Python not found"
SOLUTION: Install Python from https://www.python.org/downloads/

PROBLEM: "pip not found"
SOLUTION: python -m ensurepip --upgrade

PROBLEM: "Module not found"
SOLUTION: pip install -r requirements_modern.txt

PROBLEM: "Port 5000 already in use"
SOLUTION: Edit app_modern.py, change last line to:
         socketio.run(app, debug=True, port=5001)

PROBLEM: "Database errors"
SOLUTION: rm healthcare.db
         python setup_enhanced.py
         python add_health_goals_table.py

PROBLEM: Missing files
SOLUTION: Download from "ready-to-use-final" branch, not "main"!
