# 🏥 AI Healthcare Chatbot - Mac M2 Instructions

## ✅ Simple 3-Step Setup

### Step 1: Download
```bash
git clone -b simple-working-mac-m2 https://github.com/jumanalnagesh80-pixel/AI-Healthcare-Chatbot.git
cd AI-Healthcare-Chatbot
```

### Step 2: Install
```bash
pip3 install -r requirements_simple.txt
```

### Step 3: Run
```bash
python3 healthcare_app.py
```

## 🌐 Access the Application

Open your browser and go to:
**http://127.0.0.1:5000**

## ✨ Features

1. **Register** - Create your account
2. **Login** - Access your dashboard
3. **Chat** - Ask AI health questions
4. **Appointments** - Book doctor appointments
5. **Dashboard** - View your health summary

## 💬 Chat Examples

Try asking:
- "I have a fever, what should I do?"
- "How to manage high blood pressure?"
- "I have a headache"
- "Book an appointment"

## 🛑 To Stop

Press `Ctrl+C` in the terminal

## 📝 Notes

- All data is stored in `healthcare_simple.db`
- No complex setup required
- Works perfectly on Mac M2
- No username field issues
- No template errors

## ✅ Tested On

- Mac M2
- Python 3.14

## 🆘 Need Help?

If you see any errors:
1. Make sure you're in the project folder
2. Make sure Python 3 is installed: `python3 --version`
3. Make sure Flask is installed: `pip3 list | grep Flask`
