## 📥 Quick Download & Run

### Option 1: Download ZIP (Easiest)

1. **Go to:** https://github.com/jumanalnagesh80-pixel/AI-Healthcare-Chatbot
2. **Switch to branch:** `ready-to-use-final` (dropdown near top left)
3. **Click:** Green "Code" button → "Download ZIP"
4. **Extract** the ZIP file
5. **Open terminal** in that folder
6. **Run:**
   ```bash
   pip install -r requirements_modern.txt
   ```

## 🔍 Troubleshooting

### Missing Python?
Install from: https://www.python.org/downloads/

### Port 5000 in use?
Edit `app_modern.py`, last line:
```python
socketio.run(app, debug=True, port=5001)  # Change to 5001
```

### Module errors?
```bash
pip install -r requirements_modern.txt
```

### Database errors?
```bash
rm healthcare.db
python setup_enhanced.py
python add_health_goals_table.py
```
